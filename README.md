# nodejs-web-preact-typescript-router-ssg-template

## Description
A template for the preact framework a superset for react.

Feature
- router
- ssg prerender

## Tech stack
- npm
- preact
- ssg

## Docker stack
- node

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
